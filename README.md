### Project Description
An AI-powered cloud monitoring system that automatically 
detects anomalies using Isolation Forest machine learning 
algorithm and sends automated email alerts via Gmail SMTP.

### Technologies Used
- Python 3.10+
- Flask
- Scikit-learn (Isolation Forest)
- psutil
- Pandas and NumPy
- SQLite
- Chart.js
- Gmail SMTP (smtplib)

### How to Run
1. pip install -r requirements.txt
2. python app.py
3. Open browser: http://localhost:5000
4. Login with: admin / admin123

### Features
- Real-time CPU Memory Disk Network monitoring
- AI anomaly detection using Isolation Forest
- NORMAL WARNING CRITICAL classification
- Automated Gmail email alerts
- 6-page professional dashboard
- Incident log with resolve workflow
- Secure SHA-256 password hashing
